# Explicit Trade

A modern trading platform dashboard built with React, Vite, and Tailwind CSS.

## Project Structure

- `src/components`: Reusable UI components and trading-specific components.
- `src/pages`: Application pages (Dashboard, Trade, Wallet, etc.).
- `src/lib`: Utility functions, types, and state management.

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
